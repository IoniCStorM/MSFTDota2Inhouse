Game

. Please report the result of the game honestly.
. Game result will be recorded when more than 50% of players have voted the game result.

Mode

. Game mode will be Captains Mode, with the highest MMR player being the default captain.
. The captain can decide to let a different player to draft.
. Game will be hosted by the player who has the most played games.
. The private lobby should have lobby name: "MSFT Dota2" with a random password given by the system.
. The default lobby region is US West, unless every player agrees to use a different region.

Player Disconnection

. The team which has the player DC'ed can request pause for up to 10 minutes
. After the pause time is used up, the team with more remaining member can resume the game.
. If a player is DC'ed before any of high-ground tower is destroyed, the team with fewer members can request the game to be ended, and record the game as "Not finished"
. If a player is DC'ed with a high-ground tower is destroyed, the team with more high-ground towers left can request the game to be ended, and record the game as "Not finished"
